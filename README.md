# mdias-oled

A vibrant OLED-friendly theme for [Ghostty](https://ghostty.org) with pure black background, marine green accents, and hot pink highlights.

## Features

- **Pure black background** (`#000000`) - Perfect for OLED displays, saves battery and prevents burn-in
- **OLED-friendly text** - Soft gray (`#d4d4d4`) instead of harsh white to reduce eye strain
- **Vibrant marine green cursor** - Easy to spot on black
- **Hot pink highlights** - Bold accent that pops

## Installation

1. Download the theme file:
   ```bash
   curl -o ~/.config/ghostty/themes/mdias-oled \
     https://raw.githubusercontent.com/mcdays94/ghostty-theme-mdias-oled/main/themes/mdias-oled
   ```

2. Add to your Ghostty config (`~/.config/ghostty/config`):
   ```
   theme = mdias-oled
   ```

3. Restart Ghostty.
